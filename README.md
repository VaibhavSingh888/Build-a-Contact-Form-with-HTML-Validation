Elevate Labs - Web Development Internship 

Task 5: Build a Contact Form with HTML Validation 

Project Description
The objective of this project was to develop a fully functional and validated contact form using standard HTML5 and CSS. The implementation focuses on ensuring a clean, modern, and accessible user interface while utilizing native browser validation techniques to ensure data integrity without the immediate need for JavaScript.

Features Implemented

Semantic Structure: Built the form using appropriate HTML elements including <form>, <input>, <textarea>, and <button>.


Input Optimization: Applied specific input types such as email, tel, and text to ensure correct data entry and keyboard layouts.


HTML5 Validation: Integrated validation attributes including required, pattern, and maxlength to enforce field requirements and formatting rules.


Accessibility: Every input field is accompanied by a <label> element linked via the for attribute to enhance accessibility for screen readers.


Logical Organization: Form elements are grouped in a logical order to improve user flow and ease of completion.


Custom CSS Styling: Utilized CSS to create a professional and clean User Interface (UI).


Browser Feedback: Configured the form to utilize default browser validation messages for clear user feedback.

Tools Used

Primary Tools: VS Code and Browser DevTools.
